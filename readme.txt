= What does this plugin do? =

* Show compatibility information for each PlusWP product

= Who is this Plugin ideal for? =
* Anyone who is interested in a PlusWP product and is not sure whther its setup is compatible with it.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. From your WordPress Admin Dashboard go to: Plugins > Add New
1. Search for "PlusWP Check"
1. Locate the "PlusWP Check" plugin by PlusWP and click the Install Now button.
1. Activate the plugin labeled "PlusWP Check".
1. You're done!
