= What does this plugin do? =

* Show compatibility information for each KnitKode product

= Who is this Plugin ideal for? =
* Anyone who is interested in a KnitKode product and is not sure whther its setup is compatible with it.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. From your WordPress Admin Dashboard go to: Plugins > Add New
1. Search for "KnitKode Check"
1. Locate the "KnitKode Check" plugin by KnitKode and click the Install Now button.
1. Activate the plugin labeled "KnitKode Check".
1. You're done!
